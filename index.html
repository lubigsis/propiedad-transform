Propiedad transform - mis apuntes

(Posicionar el cursor sobre el unicornio)

transform:rotate();

Gira el elemento respecto a su posición original. El ángulo de giro viene indicado en el parámetro.

Unidades: turn (0 a 1), deg (0° a 360°), grad (0 a 400), rad (0 a 6.28319).

Ejemplo 1:

transform: rotate(45deg);
transition: transform 1s linear;

transform:rotate();

Ejemplo 2:

transform: rotate(-45rad);
transition: transform 6s linear;

transform:rotateX();

Ejemplo 3:

transform: rotateX(60deg);
transition: transform 1s linear;

transform:rotateY();

Ejemplo 4:

transform: rotateY(50deg);
transition: transform 1s linear;

transform:rotateZ();

Ejemplo 5:

transform: rotateZ(80deg);
transition: transform 1s linear;

transform-origin();

Indica cuál será el punto de origen desde el que se transforma el elemento. Tiene 2 valores, el primero indica la coordenada 'x' del origen: transform-origin:[porcentaje | medida | left | center | right] El segundo valor indica la coordenada 'y', que puede expresarse en: transform-origin:[porcentaje | medida | top | center | bottom]

Ejemplo 6:

transform: rotate(30deg);
transform-origin: right bottom;
transition: transform 1s linear;

transform-origin();

Ejemplo 7:

transform: rotate(30deg);
transform-origin: center bottom;
transition: transform 1s linear;

transform-origin();

Ejemplo 8:

transform: rotate(30deg);
transform-origin: left bottom;
transition: transform 1s linear;

transform-origin();

Ejemplo 9:

transform: rotate(30deg);
transform-origin: center top;
transition: transform 1s linear;

transform-origin();

Ejemplo 10:

transform: rotate(30deg);
transform-origin: left top;
transition: transform 1s linear;

transform-origin();

Ejemplo 11:

transform: rotate(30deg);
transform-origin: right top;
transition: transform 1s linear;

transform-origin();

Ejemplo 12:

transform: rotate(360deg);
transform-origin: top left;
transition: transform 5s linear;

transform-origin();

Ejemplo 13:

transform: rotate(360deg);
transform-origin: center center;
transition: transform 1s linear;

transform-origin();

Ejemplo 14:

transform: rotate(30deg);
transform-origin: 10% 15%;
transition: transform 1s linear;

transform: translate();

Cambia la posición del elemento (traslación), moviéndolo hacia donde indican los parámetros 'x', 'y'.

Ejemplo 15:

transform: translate(500px);
transition: transform 6s linear;

transform:translate();

Ejemplo 16:

transform: translate(-500px);
transition: transform 6s linear;

transform:translate();

Ejemplo 17:

transform: translateX(200px);
transition: transform 6s linear;

transform:translate();

Ejemplo 18:

transform: translateY(-150px);
transition: transform 6s linear;

transform:translate();

Ejemplo 19:

transform: translate(70px, 30px);
transition: transform 5s linear;

transform: skew();

Inclina el elemento. El primer parámetro indica la inclinación de los lados verticales, y el segundo, los horizontales. Se usa el deg o rad.

Ejemplo 20:

transform: skew(20deg);
transition: transform 1s linear;

transform:skew();

Ejemplo 21:

transform: skew(-20deg);
transition: transform 1s linear;

transform:skew();

Ejemplo 22:

transform: skew(80deg);
transition: transform 5s linear;

transform:skewX();

Ejemplo 23:

transform: skewX(50deg);
transition: transform 1s linear;

transform:skewY();

Ejemplo 23:

transform: skewY(50deg);
transition: transform 1s linear;

transform: scale();

Aumenta o reduce de tamaño el elemento en proporción (no es como la prop. with). Los ejes horizontal y vertical se controlan independientemente. Acepta valores positivos, negativos y decimales. Su cambio de tamaño no afecta a otros elementos que estén a lado.

Ejemplo 25:

transform: scale(2);
transition: transform 1s linear;

transform:scale();

Ejemplo 26:

transform: scale(0.5);
transition: transform 1s linear;

matrix();

Combina todos los métodos anteriores. Utiliza 6 parámetros.

Para evitar sobreescritura, se pueden emplear múltiples transformaciones separándolas mediante un espacio.

Ejemplo 27:

transform: matrix(0, -1, -1.22, -0.88, 11, 11);
transition: transform 3s linear;

transform:matrix();

(Combinación más simple)

Ejemplo 28:

transform: scale(0.5);
transition: transform 1s linear;

transform:matrix();

(Combinación más simple)

Ejemplo 29:

transform: scale(1.2) skew(20deg);
transition: transform 1s linear;
